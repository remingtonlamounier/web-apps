Projetos Este é o centro de criação dos seus projetos.

Passo 1

Para criar um projeto clique no botão e informe uma descrição para o projeto. Por exemplo, supondo que você gostaria de um site para divulgar seu trabalho:

Descrição: Como consultor gostaria de um site para divulgar os tipos de consultoria que presto as empresas.

Passo 2

Logo em seguida você deve informar quais as funcionalidades que você espera no seu projeto respondendo apenas 3 perguntas. Vamos continuar o exemplo acima e descrever uma funcionalidade:

Ator (quem vai executar esta ação): Administrador

Descrição (o quê o ator gostaria de fazer): Gostaria de publicar toda semana fotos das minhas últimas consultorias.

Objetivo (o quê o ator espera com esta funcionalidade): Para que os visitantes vejam os frutos do meu trabalho.

Passo 3

Uma funcionalidade pode conter critérios de aceitação, neste caso você também deve informá-los. Veja um exemplo baseado na funcionalidade do passo 2:

Critério: O sistema deve bloquear um visitante acessar esta área do site.
