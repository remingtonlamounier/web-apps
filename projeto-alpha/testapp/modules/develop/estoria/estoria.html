Criando estórias

Clique no botão abaixo, em seguida descreva de forma simples a estória do usuário de acordo com o quê ele deseja. Particione os desejos do usuário em várias estórias para facilitar as sprints e se necessário informe os critérios de aceitação. Por exemplo:

Desejo do usuário

Como consultor gostaria de um site para divulgar o meu trabalho. O site precisa ter uma área administrativa para que eu possa publicar fotos e escrever artigos, assim os visitantes poderam ver o trabalho que presto para as empresas.

Estória 1

Ator: Administrador
Descrição: Publicar fotos no site
Objetivo: Para que os visitantes do site conheçam o meu trabalho.
